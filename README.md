## 🌐 Verkkosivusto

Tämä on Tornion VanneFixaus -yrityksen virallinen verkkosivusto.

## 🛠️ Teknologiat

- React + TypeScript
- Vite
- Tailwind CSS
- GSAP (animaatiot)
- shadcn/ui

## 📋 Vaatimukset

- Node.js 20+
- npm

## 🚀 Paikallinen kehitys

```bash
# Asenna riippuvuudet
npm install

# Käynnistä kehityspalvelin
npm run dev

# Rakenna tuotantoversio
npm run build
```
